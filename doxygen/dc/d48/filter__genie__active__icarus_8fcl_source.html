3 # Purpose: Filters out all events with no neutrino interaction in active volume.

8 #
9 # The events in the input file are filtered with the requirement that at least
10 # one neutrino interaction happens within the active volume of the detector.
11 # See `FilterNeutrinosActiveVolume` documentation for more details.
