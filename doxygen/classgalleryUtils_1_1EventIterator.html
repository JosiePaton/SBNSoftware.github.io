Detailed Description

Iterator for an event sequence.

It can be used directly as:
    
    for event in EventIterator(event): ...
    
or it can be plugged in directly in the gallery.Event class, making the latter
be iterable:
    
    for event in event: ...
    
(or `for evt in event: ...`).
