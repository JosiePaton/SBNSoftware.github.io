Detailed Description

Prints the content of all the partidle IDs on screen.

This analyser prints the content of all the particle IDs into the LogInfo/LogVerbatim stream.

Configuration parameters

  • ParticleIDModuleLabel (string): label of the producer used to create the anab::ParticleID collection
  • OutputCategory (string, default: "DumpParticleIDs"): the category used for the output (useful for filtering)
