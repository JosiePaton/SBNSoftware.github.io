Detailed Description

Prints the content of all the vertices on screen.

This analyser prints the content of all the vertices into the LogInfo/LogVerbatim stream.

Configuration parameters

  • VertexModuleLabel (art::InputTag, mandatory): label of the producer used to create the recob::Vertex collection to be dumped
  • OutputCategory (string, default: "DumpVertices"): the category used for the output (useful for filtering)
  • PrintHexFloats (boolean, default: false): print all the floating point numbers in base 16
