40 /**
41  * @brief Base interface for gate builders.
42  * @see `icarus::trigger::DynamicTriggerGateBuilder`,
43  * `icarus::trigger::FixedTriggerGateBuilder`
44  *
45  * This base class provides a skeleton building algorithm that can be customised
46  * by templates.
47  * The allowed customization includes what to do when a threshold is crossed
48  * in a gate.
49  *
50  * Note that actions are performed only when the sample crosses a threshold.
51  * The algorithm keeps track at each time of which are the thresholds enclosing
52  * the signal level, and if the level crosses one of them, the gates associated
53  * to those thresholds, and only them, are offered a chance to react.
54  */
