Detailed Description

Prints the content of all the PCA axis object on screen.

This analyser prints the content of all the principal component axis object into the LogInfo/LogVerbatim stream.

Configuration parameters

  • PCAxisModuleLabel (art::InputTag, mandatory): label of the producer used to create the recob::PCAxis collection to be dumped
  • OutputCategory (string, default: "DumpPCAxes"): the category used for the output (useful for filtering)
  • PrintHexFloats (boolean, default: false): print all the floating point numbers in base 16
