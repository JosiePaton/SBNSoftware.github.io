Detailed Description

Creates some dummy hits.

The produced tracks have completely dummy content.

Configuration parameters

  • hits (input tag, mandatory): the data product to read the hits from
  • hitsPerTrack (list of unsigned integers, mandatory): number of hits for each produced track. If there are hits left after all the tracks specified here have been created, an additional track with all those hits is created. If there are not enough hits, an exception is thrown.
