Detailed Description

Prints the content of all the hits on screen.

This analyser prints the content of all the hits into the LogInfo/LogVerbatim stream.

Configuration parameters

  • HitModuleLabel (string): label of the producer used to create the recob::Hit collection
  • OutputCategory (string, default: "DumpHits"): the category used for the output (useful for filtering)
  • CheckWireAssociation (string, default: false): if set, verifies that the associated wire are on the same channel as the hit
  • CheckRawDigitAssociation (string, default: false): if set, verifies that the associated raw digits are on the same channel as the hit
