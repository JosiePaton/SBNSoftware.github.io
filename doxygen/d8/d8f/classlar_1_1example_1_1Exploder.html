Detailed Description

A buggy module.

This module executes a number of iterations. It does something in some of those iterations, depending on the module configuration.

Configuration

  • NIterations (integer, default: 10): number of iterations executed
  • OutOfRange (integer, optional): if specified, the module throws a std::out_of_range exception at the specified loop number
