Detailed Description

template<typename T, typename = void>
struct icarus::details::KeyValuesData::Item::StringConverter< T, typename >

Converter class for interpreting strings as type T.

Template Parameters
Ttype to convert strings into

This class is designed to allow handling special types via specialization. The standard implementation uses from_chars() (including the one from std, but also any pulled in by C++ name lookup) to perform the conversion, and returns a empty optional on failure. Specializations may decide to throw an exception instead (type ConversionFailed or derivatives are suggested).
